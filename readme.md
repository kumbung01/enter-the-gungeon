# 구현 시 고려 사항
- 총알(탄막) 어떻게 처리할 건지?
	- 매번 생성-삭제 는 프레임 낭비 심함
	- 총알 재활용
- 픽셀 그리는 방법?
	- 텍스쳐가 1x1 픽셀임. 픽셀을 사각형으로 키워야함.
- 총 회전?
	- 회전 후 다시 pixelate 하는 것으로 보임.
	- winapi setpixel 로 구현 가능한지.
- 블루밍?
	- 조명이나 총알 빛
